StartUp instructions

Access the website on heroku :
To login as student visit - https://tohms.herokuapp.com/project/#/home
To login as TA visit - https://tohms.herokuapp.com/project/#/taHome

Alternatively, use git to clone and run on local system:
1. Clone the git repository mentioned above using - git clone https://github.ccs.neu.edu/keerthi/cs5340-project.git
2. This is a private repository, you won't have access to it unless we add you as collaborator. Please email your git.ccs.neu.edu id to us at - keerthi@ccs.neu.edu or kmcarol26@ccs.neu.edu as soon as possible for I7 testing.
3. Once you obtained access and successfully cloned the project, on the terminal switch to cs5340-project folder by typing : cd cs5340-project
4. run the command : npm install
5. run the command : mongod
6. On another terminal (without closing the first one), run the command : node server.js
7. To login as student - Go to localhost:3000/project/#/home . To login as TA - Go to localhost:3000/project/#/taHome .

Breifing from P4

Hi, we are designing a Teaching Assistant Office Hours Management System (TOHMS). The purpose of this application is to allow students to book appointments with the TAs for their office hours visits. The TAs in turn can use the application to be give their availability and see information about booked appointments. Informed consent: We are conducting a study to find out what people think about this. We will not record or publish any information with your name. This is for a course we are taking in Human-Computer Interaction under Prof. David Sprague. Your participation is voluntary, and you can stop anytime and ask that your data not be used. It should take about 20 minutes to 30 minutes. Can you help us out with this? Awesome. The purpose of today's session is for you to help us figure out how to make this web interface more user-friendly before we finish developing it. But believe it or not, we are not going to use a computer. As you'll see, we have actually created paper versions of the screens, and my friend Keerthi will be playing the computer. If you want to do something, you'll need to interact with the prototype just as you would on a computer. Use your finger to click on buttons. These pieces of tape indicate places where you can type something in, and here's your keyboard (give pen). It's OK to write on this. Your job is to tell us what makes sense, what's confusing, whether it works the way you'd expect it to. Keep in mind that we're testing the interface--we're not testing you--so if you run into any problems it's not your fault and it means that there's something we need to change. I'll be sitting next to you, and I can help you if you want. Please tell us what makes sense to you, what's confusing, and any questions that come to mind. Your questions are especially valuable, but I may not answer them right away because our goal is to change the interface, so it answers them. Remember that we're testing the interface--we're not testing you. Are you ready to start? OK, here is the first thing we'd like you to do. Take a minute to read this and let me know if it makes sense. If so, then whenever you're ready please show us what you would do first.



Tasks

Task 1
You are a student at NEU and have taken two courses this semester. You and your friend have a question to ask the Teaching Assistant In HW-2 of Database Management Systems course. You are having difficulty designing the database for the Assignment problem, specifically in terms of normalizing the tables. Use the TOHMS(TA Office Hours Management Application) to look for the next TA office hour session in the month of November and make an appointment for you and your friend .

Task 2
You are a student at NEU. You have a question about File uploading in HW-1 in Database Management Systems course. Your friend told you that he made an appointment on TOHMS for the same question already for the 22nd November, 2018 TA hour session. Use TOHMS to join the same.

Task 3
You are a teaching assistant for Database Management System course at NEU. You want to prepare for the next TA office hour session (on 22nd November, 2018). View all the student questions for the same using TOHMS.
